# minesweeper
Classic version of minesweeper, built entirely in vanilla Javascript, HTML, and CSS.

# How to play:
Left click to reveal tile, right click to mark tile as a mine. Change difficulty via dropdown at the bottom. Reset game by pressing the button with the smiley face. You can reveal the mine locations by using the fail() function in console, and simulate a win by using win() in the console. Happy hunting!
